# JoffeGarage

## BOM
In addition to the 3D printed parts, you need:

- Hinges
- Various nuts and bolts
- Arduino UNO
- Stepper motor, NEMA-17 200steps/rev
- Stepper driver
- Ultrasonic sensor, HC-SR04
- Light dependent resistor
- 10k resistor
- Wires

## Arduino wiring
| Pin nr | Encoder |
|--------|---------|
| 3      | A       |
| 4      | B       |
| 5      | GND     |
| 6      | BTN     |
